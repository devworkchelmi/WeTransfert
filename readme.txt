WeTransfert
Pages : 
	connexion : formulaire de connexion avec mdp email 
		    bouton creation de compte

	Création de compte : formulaire de création de compte : mail, mdp, nom, prénom, numéro de tel, date de naissance
			     bouton d'envoie, bouton de suppression

	modifier son profile : afficher les informations du profile : mail, mdp, nom, prénom, numéro de tel, date de naissance
			       bouton pour les modifier, bouton de déconnexion

	
	téléchargement, suppression, envoie fichier : affichage des fichiers déjà envoyer avec bouton de suppression et de téléchargement
						      bouton pour mettre un fichier bouton pour l'envoyer
						      bouton de déconnexion



fonctionnalités : 

	connexion : vérification adresse mail et mot de passe
	 
	Création de compte : confirmation du mdp et mdp (saler) haché
			     stockage base de donner ou fichier
			     email unique

	modifier son profile : confirmation du mdp et mdp (saler) hashé
			     stockage base de donner ou fichier
			     email unique

	 envoi de fichier : pas de .php
			  nom de fichier = hash de l'email de l'utilisateur
			  fichier < 20mo
			  bonus : enregistrer le fichier sous un nom différent et unique tout en retenant le nom de base
	
	téléchargement :  récupérer les fichier déjà envoyé et les afficher (que pour la personne connecter pas les autres utilisateur)
			  permettre le téléchargement de ces fichiers
			  Compter le téléchargement de chaque fichiers
			  Possibilité d'associer un fichier à une adresse mail pour que la personne qui possède cette adresse mail puisse aussi télécharger le fichier
	
	suppression : possibilité de supprimer un fichier envoyé (avec une confirmation)
