WeTransfert,site d'enregistrement et d'envoi de fichiers
Pages : 
	Connexion : formulaire de connexion avec deux champs mot de passe et adresse mail 


	Création de compte : formulaire de création de compte : mail, nom, prénom, date de naissance 
						avec double confirmation de mot de passe 


	Page profil :  possibilité de changer d'adresse mail et de mot de passe

	
	Page de téléchargement, suppression et envoi de fichier : 
	affichage d'une liste des fichiers déjà envoyé avec bouton de suppression et de téléchargement
	possibilité de charger un autre fichier vers le serveur (enregistrer un nouveau fichier)


Fonctionnalités et précision : 
Impossibilité d'enregistrer plusieurs fois un fichier qui a le même nom (les fichiers sont enregistrés sous un autre nom, côté serveur)

Si profil connu, la page de création de compte, affiche un message de bienvenue.

Possibilité de se déconnecter via la page profil ou le bouton sur le header.
